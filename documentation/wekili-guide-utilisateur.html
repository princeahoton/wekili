1. Premiers pas — Créer votre compte

Wekili est 100 % gratuit. Aucune carte bancaire n'est requise. Votre inscription prend moins de 2 minutes.

1.1 Inscription par email

1
Accéder à la page d'inscription
Rendez-vous sur wekili.vercel.app puis cliquez sur le bouton « S'inscrire » en haut à droite, ou directement sur /register.
2
Remplir le formulaire
Saisissez votre prénom, nom, pays d'origine, adresse email et un mot de passe (minimum 8 caractères). Acceptez les conditions générales.
3
Vérifier votre email
Un email de confirmation est envoyé automatiquement. Cliquez sur le lien qu'il contient pour activer votre compte. Vérifiez vos spams si vous ne le recevez pas dans les 5 minutes.
4
Se connecter
Une fois l'email vérifié, connectez-vous avec votre email et mot de passe sur /login.
Conseil : Utilisez une adresse email active que vous consultez régulièrement. Wekili vous enverra des alertes sur les deadlines des bourses et confirmations importantes.

1.2 Connexion avec Google

Sur la page de connexion ou d'inscription, cliquez sur le bouton « Continuer avec Google ». Une fenêtre Google s'ouvre pour choisir votre compte. Aucun mot de passe Wekili n'est nécessaire — votre compte est créé automatiquement à la première connexion.

Avantage Google : Votre email est automatiquement vérifié, vous accédez à la plateforme immédiatement sans attendre un email de confirmation.

1.3 Connexion par téléphone (SMS)

1
Entrer votre numéro
Sur la page de connexion, choisissez l'onglet « Téléphone » et saisissez votre numéro au format international (ex : +22670000000).
2
Recevoir le code SMS
Un code à 6 chiffres est envoyé par SMS. Il est valable 10 minutes.
3
Saisir le code
Entrez le code reçu dans le champ prévu. Vous êtes connecté directement.

1.4 Mot de passe oublié

1
Sur la page de connexion, cliquez sur « Mot de passe oublié ».
2
Entrez votre adresse email et cliquez sur « Envoyer ». Un lien de réinitialisation valable 1 heure vous est envoyé.
3
Cliquez sur le lien dans l'email, entrez votre nouveau mot de passe et confirmez.
Important : Si vous vous êtes inscrit avec Google ou par téléphone, vous ne pouvez pas réinitialiser un mot de passe — utilisez la même méthode de connexion qu'à l'inscription.

3. Compléter votre profil académique

Le profil est la base de toute l'intelligence de Wekili. Plus il est complet, plus vos analyses, correspondances bourses et recommandations seront précises. Accès via le menu « Mon profil » ou le bouton sur le tableau de bord.

Sections du profil

Informations personnelles
  • Nationalité — important pour l'éligibilité aux bourses
  • Pays de résidence actuel
  • Date de naissance — certaines bourses ont un âge maximum
  • Téléphone
Parcours académique
  • Niveau d'études actuel (Licence, Master, Doctorat)
  • Domaine d'études
  • Établissement actuel
  • Moyenne générale (sur 20)
Langues
  • Langue principale + niveau (A1 à C2)
  • Certification (DELF, IELTS, TOEFL, TCF…)
  • Langue secondaire (optionnel)
Projet d'études
  • Pays cibles (France, Canada, Belgique, Allemagne, Royaume-Uni)
  • Niveau visé (Master, Doctorat…)
  • Domaine visé
  • Budget mensuel disponible
Expérience et motivation
  • Expérience professionnelle
  • Activités extra-académiques
  • Lettre de motivation courte (décrit votre projet)
Logistique
  • Date d'arrivée souhaitée
  • Budget mensuel logement
Important : Remplissez au minimum ces champs pour obtenir une analyse pertinente : niveau d'études, domaine, moyenne, langue principale + niveau, pays cibles. Sans ces informations, l'IA ne peut pas calculer votre éligibilité aux bourses.

Enregistrer votre profil

Cliquez sur le bouton « Enregistrer » en bas de la page. Une notification verte confirme la sauvegarde. Vous pouvez revenir modifier votre profil à tout moment.

4.2 Comment uploader un document

1
Accéder à la section Documents
Depuis le menu de navigation, cliquez sur « Documents ».
2
Si un PIN est actif — déverrouiller
Si vous avez configuré un PIN de sécurité, entrez-le pour accéder à vos documents. Sinon, la liste s'affiche directement.
3
Choisir le type et le fichier
Cliquez sur « Ajouter un document », sélectionnez le type dans la liste déroulante, puis choisissez votre fichier (taille max : 10 MB).
4
Confirmer l'upload
Le fichier est téléchargé sur nos serveurs sécurisés. Une barre de progression et une confirmation apparaissent. L'action est enregistrée dans le journal d'activité.
Attention : Ne supprimez pas un document si vous avez une analyse en cours. Cela peut affecter les résultats. Si vous devez remplacer un document, uploadez la nouvelle version puis supprimez l'ancienne.

4.3 Protéger vos documents avec un PIN

Le PIN ajoute une couche de sécurité : même si quelqu'un accède à votre compte, il ne peut pas consulter vos documents sans le PIN.

1
Dans la section Documents, cliquez sur « Configurer un PIN ».
2
Entrez un code à 4 chiffres et confirmez-le. Cliquez sur « Créer le PIN ».
3
À chaque visite de la page Documents, ce PIN vous sera demandé. En cas d'oubli, cliquez sur « PIN oublié » — un code de réinitialisation vous sera envoyé par email.

4.4 Journal d'activité

Chaque action (upload, suppression, accès) est enregistrée avec la date, l'heure et le type d'action. Consultez-le en cliquant sur « Journal » dans la barre latérale de la page Documents. Cela vous permet de détecter tout accès suspect.

5. Analyse IA de votre dossier

L'analyse IA est la fonctionnalité centrale de Wekili. Elle évalue votre profil complet et vous donne une vision claire de votre situation, de vos points forts et des bourses pour lesquelles vous êtes le plus éligible.

Prérequis : Pour obtenir une analyse pertinente, assurez-vous d'avoir complété votre profil académique (niveau, domaine, moyenne, langues, pays cibles) et d'avoir uploadé au moins 2 documents.

5.1 Lancer une analyse

1
Accédez à la page « Analyse » depuis le menu de navigation.
2
Cliquez sur le bouton « Lancer l'analyse de mon dossier ». L'IA examine votre profil, vos documents et les données des bourses. Cette opération prend environ 15 à 30 secondes.
3
Les résultats s'affichent directement sur la page, structurés en plusieurs onglets.

5.2 Comprendre les résultats

Score global (sur 100)
Évalue la solidité globale de votre dossier pour étudier à l'étranger. Un score ≥ 70 indique un bon dossier. Un score < 50 signifie qu'il y a des axes d'amélioration importants.
Synthèse
Un résumé en langage clair de votre profil, rédigé par l'IA. Il pointe les éléments les plus importants de votre dossier.
Points forts / Faiblesses
Liste structurée des atouts de votre dossier (ex : moyenne élevée, certification DELF B2) et des points à améliorer avant de candidater.
Programmes recommandés
Sélection de programmes universitaires dans vos pays cibles, adaptés à votre profil, avec la raison de chaque recommandation.
Estimation des chances
Pour chaque pays cible, une estimation en pourcentage et niveau (Faible / Moyen / Élevé) avec un commentaire contextuel.
Recommandations
Actions concrètes à entreprendre pour améliorer votre profil et augmenter vos chances (ex : passer le DELF B2, améliorer la LM, ajouter une lettre de recommandation).

5.3 Relancer une analyse

Vous pouvez relancer une analyse après avoir amélioré votre profil ou uploadé de nouveaux documents. L'historique de toutes vos analyses est conservé — vous pouvez comparer votre progression dans le temps.

Stratégie : Commencez par lancer une première analyse pour identifier vos faiblesses. Corrigez-les (profil, documents, certification de langue), puis relancez pour mesurer l'amélioration de votre score.

6. Bourses — Trouver et comprendre votre éligibilité

Wekili recense les principales bourses pour étudiants africains souhaitant étudier en France, Canada, Belgique, Allemagne et au Royaume-Uni. Chaque bourse affiche votre score d'éligibilité personnalisé (après analyse IA).

6.1 Explorer le catalogue

Accédez à la page « Bourses » depuis le menu. Vous pouvez filtrer par :

  • Pays — France, Canada, Belgique, Allemagne, Royaume-Uni
  • Niveau — Licence, Master, Doctorat
  • Domaine — Sciences, Droit, Médecine, etc.
  • Type de financement — Bourse complète ou partielle

8. Lettre de motivation assistée par IA

L'IA peut vous aider de deux façons : générer une lettre de motivation complète à partir de votre profil, ou corriger une lettre que vous avez déjà rédigée.

Ces fonctionnalités se trouvent dans la page « Analyse », onglet « Lettre de motivation » dans la barre latérale.

8.1 Générer une lettre avec l'IA

1
Choisir l'université cible
Indiquez le nom de l'université et le programme visé. Plus vous êtes précis, meilleure sera la lettre générée.
2
Générer
Cliquez sur « Générer une première version avec l'IA ». L'IA utilise votre profil (parcours, motivation, pays cibles) pour rédiger une lettre personnalisée en ~20 secondes.
3
Réviser et personnaliser
La lettre générée est un point de départ solide. Modifiez-la dans l'éditeur de texte pour y ajouter vos anecdotes personnelles, vos projets spécifiques, vos expériences propres. Une bonne lettre reste personnelle.
4
Sauvegarder la version
Cliquez sur « Sauvegarder ». Chaque version est conservée avec un numéro (v1, v2…) pour retrouver l'historique.

8.2 Faire corriger votre lettre par l'IA

1
Collez ou rédigez votre lettre dans l'éditeur de texte.
2
Cliquez sur « Faire corriger par l'IA ». L'analyse dure ~15 secondes.
3
Vous recevez : un score sur 100, la liste des points forts, les points à améliorer et une version corrigée complète. Comparez les deux versions et intégrez les corrections qui vous conviennent.
Conseil : Pour une lettre de qualité optimale, combinez les deux approches : générez une base avec l'IA, personnalisez-la avec vos expériences propres, puis faites-la corriger pour affiner le score.
Rappel : La lettre de motivation générée est un support de travail. Vérifiez toujours que le contenu reflète fidèlement votre parcours réel avant de la soumettre à une université ou organisme de bourse.

9. CV — Analyse et correction IA

L'IA analyse votre CV et le corrige en tenant compte des normes spécifiques au pays que vous ciblez. Un CV pour la France n'a pas le même format qu'un CV pour le Royaume-Uni.

Ces fonctionnalités se trouvent dans la page « Analyse », onglet « CV » dans la barre latérale.

9.1 Importer votre CV

Deux options :

  • Uploader un PDF — cliquez sur le bouton d'import. Le texte est extrait automatiquement du fichier.
  • Coller le texte — collez directement le contenu de votre CV dans l'éditeur.

9.2 Choisir le pays cible

Sélectionnez le pays pour lequel vous candidatez. L'IA adapte l'analyse aux normes locales :

PaysSpécificités du CV analysées
FrancePhoto recommandée, état civil, expériences en ordre chronologique inversé, compétences linguistiques détaillées
Royaume-UniPas de photo ni de date de naissance, 2 pages max, Personal Statement en introduction
CanadaFormat Résumé ou CV académique, pas de photo, réalisations chiffrées valorisées
AllemagneLebenslauf — photo obligatoire, date et lieu de naissance, très structuré
BelgiqueSimilaire à la France, photo optionnelle, format court valorisé

9.3 Lancer l'analyse

1
Sélectionnez le pays cible dans le menu déroulant.
2
Cliquez sur « Analyser et corriger mon CV ». L'analyse prend ~20 secondes.
3
Consultez les résultats : score, points forts, corrections prioritaires, sections manquantes, norme du pays, et version corrigée complète.
4
Cliquez sur « Sauvegarder » pour conserver cette version dans l'historique.

10. Logement — Préparer votre arrivée

La section Logement centralise toutes les ressources pour trouver un logement dans votre pays de destination : plateformes recommandées, aides financières, estimation de budget et checklist de démarches.

10.1 Configurer votre recherche

En haut de la page, renseignez :

  • Ville de destination (ex : Paris, Montréal, Berlin…)
  • Budget mensuel en euros
  • Statut de recherche — votre avancement actuel

Cliquez sur « Sauvegarder » pour mémoriser ces informations. Le pays dans la barre latérale détermine les plateformes et aides affichées.

10.2 Plateformes recommandées

Pour chaque pays, Wekili liste les meilleures plateformes de recherche de logement avec :

  • Le type de logement (résidence universitaire, colocation, particuliers…)
  • La fourchette de prix mensuelle
  • Le niveau de priorité (haute ou normale)
  • Un conseil pratique sur comment utiliser la plateforme
  • Le lien direct vers le site
France : Commencez toujours par le CROUS (trouverunlogement.lescrous.fr) — les places sont limitées mais les prix sont les plus bas (200–400 €/mois). Faites la demande dès que vous avez votre admission.

10.3 Aides financières au logement

Wekili liste les aides disponibles dans votre pays de destination (ex : APL/CAF en France, CPAS en Belgique, bourses universitaires au Canada). Chaque aide indique le montant estimé, un lien vers le site officiel et un conseil sur la démarche à suivre.

10.4 Estimation du budget mensuel

En saisissant votre ville de destination, Wekili affiche une estimation du budget mensuel typique :

  • Fourchette de loyer (min / max)
  • Charges estimées
  • Coût de vie mensuel moyen
  • Total estimé par mois

Les données couvrent les principales villes étudiantes : Paris, Lyon, Bordeaux, Toulouse, Montréal, Québec, Berlin, Londres, Bruxelles, etc.

10.5 Checklist logement

La checklist répertorie toutes les démarches à effectuer, organisées en 5 groupes chronologiques :

  • Avant de chercher — budget, type de logement souhaité, assurance
  • Dossier de location — documents à préparer (passeport, garant, justificatifs)
  • Recherche active — inscription sur les plateformes, envoi de candidatures
  • Visite et signature — état des lieux, lecture du bail, dépôt de garantie
  • Après signature — demande APL, abonnements (eau, électricité, internet)

Cochez chaque étape au fur et à mesure. Une barre de progression indique votre avancement global.

10.6 Guide IA personnalisé

Cliquez sur « Guide IA » pour recevoir des recommandations personnalisées sur votre situation logement. L'IA prend en compte votre ville, votre budget et votre statut pour vous suggérer les meilleures options et priorités.

11. Paramètres du compte

Accédez aux paramètres via le menu de navigation → « Paramètres ». Cette section vous permet de gérer tous les aspects de sécurité et de personnalisation de votre compte.

11.1 Modifier vos informations

Dans l'onglet « Mon compte », vous pouvez modifier :

  • Votre prénom et nom
  • Votre pays d'origine
  • Votre photo de profil (URL d'une image)

Cliquez sur « Enregistrer » pour confirmer les modifications.

11.2 Changer votre mot de passe

1
Dans l'onglet « Sécurité », cliquez sur « Changer le mot de passe ».
2
Entrez votre mot de passe actuel, puis le nouveau mot de passe (minimum 8 caractères) et confirmez-le.
3
Cliquez sur « Mettre à jour ». Une confirmation est affichée.

11.3 Activer la double authentification (2FA)

La 2FA ajoute une sécurité supplémentaire : même si quelqu'un connaît votre mot de passe, il ne pourra pas se connecter sans votre téléphone.

1
Dans l'onglet « Sécurité », cliquez sur « Activer la 2FA ».
2
Installez une application d'authentification sur votre téléphone : Google Authenticator, Authy ou Microsoft Authenticator.
3
Scannez le QR code affiché avec l'application. Un code à 6 chiffres s'affiche, renouvelé toutes les 30 secondes.
4
Entrez ce code dans le champ de confirmation et cliquez sur « Confirmer ». La 2FA est maintenant active.
Important : Conservez votre téléphone accessible lors de chaque connexion. Si vous perdez accès à votre application 2FA, contactez le support Wekili pour désactiver la 2FA manuellement.

11.4 Historique des connexions

Dans l'onglet « Sessions », consultez toutes vos connexions récentes : adresse IP, appareil (navigateur + système), date et heure. Si vous voyez une connexion suspecte (pays inconnu, appareil non reconnu), changez immédiatement votre mot de passe.

11.5 Supprimer votre compte

En bas de la page Paramètres, le bouton « Supprimer mon compte » supprime définitivement toutes vos données. Cette action est irréversible. Une confirmation est demandée avant l'exécution.

Attention : La suppression de compte efface votre profil, vos analyses, vos candidatures et l'accès à vos documents. Téléchargez vos documents depuis S3 avant de supprimer votre compte si vous souhaitez les conserver.

12. Questions fréquentes (FAQ)

Wekili est-il gratuit ?
Oui, entièrement gratuit. Aucune carte bancaire n'est requise. Toutes les fonctionnalités (analyse IA, bourses, universités, LM, CV, logement) sont accessibles sans abonnement.
Mes documents sont-ils en sécurité sur Wekili ?
Vos documents sont stockés sur des serveurs AWS S3 sécurisés avec chiffrement. Seul votre compte peut y accéder. Le système de PIN ajoute une protection supplémentaire. Wekili ne partage vos documents avec aucun tiers.
Pourquoi mon score IA est-il bas ?
Plusieurs raisons possibles : profil incomplet (surtout moyenne, niveau et langue), documents manquants, ou objectif trop ambitieux par rapport à votre profil actuel. Complétez votre profil entièrement, uploadez tous vos documents et relancez l'analyse.
Puis-je candidater directement depuis Wekili ?
Non. Wekili prépare votre dossier et vous oriente, mais la candidature finale se fait directement sur les sites officiels des universités ou organismes de bourse (liens fournis dans chaque fiche). Campus France, uni-assist, UCAS sont des plateformes externes.
Je n'ai pas reçu l'email de vérification. Que faire ?
Vérifiez d'abord votre dossier spam/courrier indésirable. Si vous ne le trouvez pas après 5 minutes, sur la page de connexion, cliquez sur "Renvoyer l'email de vérification" et saisissez votre email.
Combien de fois puis-je lancer une analyse IA ?
Autant de fois que vous le souhaitez. Il est recommandé de relancer une analyse après chaque mise à jour significative de votre profil (nouveau document uploadé, niveau de langue amélioré, etc.) pour voir l'évolution de votre score.
La lettre de motivation générée est-elle directement utilisable ?
Elle constitue une excellente base de travail, mais doit impérativement être personnalisée. Les recruteurs et comités de sélection repèrent facilement les lettres génériques. Ajoutez des exemples concrets de votre parcours, vos motivations spécifiques pour l'établissement ciblé, et vos projets professionnels précis.
Mon CV est en Word. Puis-je l'analyser ?
Le module CV accepte les PDFs pour l'extraction automatique. Si votre CV est en Word (.doc/.docx), convertissez-le en PDF d'abord (Fichier → Exporter → PDF dans Word), ou copiez-collez le texte directement dans l'éditeur.
Comment changer le pays affiché dans le module Logement ?
Dans la barre latérale de la page Logement, cliquez sur le nom du pays actuel pour ouvrir le sélecteur de pays. Choisissez le pays de destination souhaité. Les plateformes et aides se mettent à jour automatiquement.
J'ai oublié mon PIN de documents. Comment le réinitialiser ?
Sur la page Documents, cliquez sur "PIN oublié" sous le formulaire de saisie. Entrez votre email et un code de réinitialisation vous sera envoyé. Entrez ce code pour définir un nouveau PIN.
Wekili fonctionne-t-il sur téléphone mobile ?
Oui, l'interface est entièrement responsive et s'adapte aux écrans mobiles. Pour une meilleure expérience lors de la rédaction de lettres de motivation ou l'analyse de CV, un ordinateur est recommandé.
Puis-je utiliser Wekili si je ne suis pas encore diplômé ?
Absolument. Si vous êtes en cours d'études (Licence par exemple), renseignez votre niveau actuel et votre moyenne actuelle. L'IA prend en compte votre situation présente et vous oriente vers les bourses et universités accessibles à votre niveau.
Wekili propose-t-il des universités hors des 5 pays actuels ?
Actuellement, Wekili couvre la France, le Canada, la Belgique, l'Allemagne et le Royaume-Uni. D'autres pays seront ajoutés dans les prochaines versions de la plateforme.
Vous avez une autre question ?
Contactez le support Wekili par email : support@wekili.app. Décrivez votre problème avec captures d'écran si possible pour une résolution plus rapide.
